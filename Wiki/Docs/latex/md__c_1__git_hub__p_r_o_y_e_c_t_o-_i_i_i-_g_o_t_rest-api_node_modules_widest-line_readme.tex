\begin{quote}
Get the visual width of the widest line in a string -\/ the number of columns required to display it \end{quote}


Some Unicode characters are \href{https://en.wikipedia.org/wiki/Halfwidth_and_fullwidth_forms}{\texttt{ fullwidth}} and use double the normal width. \href{http://en.wikipedia.org/wiki/ANSI_escape_code}{\texttt{ A\+N\+SI escape codes}} are stripped and doesn\textquotesingle{}t affect the width.
