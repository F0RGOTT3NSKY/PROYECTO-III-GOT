Return an array of concatenated elements by calling {\ttfamily fn(x, i)} for each element {\ttfamily x} and each index {\ttfamily i} in the array {\ttfamily xs}.

When {\ttfamily fn(x, i)} returns an array, its result will be concatenated with the result array. If {\ttfamily fn(x, i)} returns anything else, that value will be pushed onto the end of the result array.\hypertarget{md__c_1__git_hub__p_r_o_y_e_c_t_o-_i_i_i-_g_o_t_rest-api-node-mysql_node_modules_concat-map__r_e_a_d_m_e_autotoc_md4063}{}\doxysection{install}\label{md__c_1__git_hub__p_r_o_y_e_c_t_o-_i_i_i-_g_o_t_rest-api-node-mysql_node_modules_concat-map__r_e_a_d_m_e_autotoc_md4063}
