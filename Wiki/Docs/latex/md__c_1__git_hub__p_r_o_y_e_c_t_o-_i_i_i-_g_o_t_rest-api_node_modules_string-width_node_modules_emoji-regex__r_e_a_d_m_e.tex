{\itshape emoji-\/regex} offers a regular expression to match all emoji symbols (including textual representations of emoji) as per the Unicode Standard.

This repository contains a script that generates this regular expression based on \href{https://github.com/mathiasbynens/unicode-12.0.0}{\texttt{ the data from Unicode v12}}.
