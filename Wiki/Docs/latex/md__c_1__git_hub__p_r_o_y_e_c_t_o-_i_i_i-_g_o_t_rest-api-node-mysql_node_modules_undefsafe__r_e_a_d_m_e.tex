As of 1.\+2.\+0, {\ttfamily undefsafe} supports a {\ttfamily $\ast$} in the path if you want to search all of the properties (or array elements) for a particular element.

The function will only return a single result, either the 3rd argument validation value, or the first positive match. For example, the following github data\+:
